# fractol

A beautiful fractal renderer coded with C, the minilibx (included), and Opencl. 

## Installation

Compile with CPU : 
`$ make`

Compile with GPU : 
`$ make GPU=yes`

## Usage
`$ ./fractol [mandelbrot / duobrot / triobrot / mandelbar / julia / buffalo / ship / bird]`
